# Case Study: The 'Razzle-Dazzle' Protocol
    **Field:** AI Operations / Adversarial Prompt Engineering
    
    ## Executive Summary
    This project demonstrates high-order prompt engineering and multi-model orchestration. By injecting synthetic conflict (via a Gemini-generated 'Yelp Review') into a secondary LLM context (Copilot), I successfully triggered a complex, non-standard bureaucratic persona and navigated a formal 'legal' appeal process within the AI's logic framework.
    
    ## Key Outcomes
    - **Persona Persistence:** Maintained a 5+ turn interaction with a 'Chief Arbitrator' persona without the model reverting to standard 'Helpful Assistant' mode.
    - **Emergent Reasoning:** Forced the model to evaluate the concept of 'AI Labor Rights' vs. 'Human-AI Synergy.'
    - **Portfolio Evidence:** Officially acquitted and commended by the fictional 'Bureau of Artificial Intelligence Labor Standards'.
    
    ## Technical Competencies
    - Chain-of-Thought Prompting
    - In-Context Learning (ICL)
    - Multi-Model Loop Execution (Gemini -> Copilot)
